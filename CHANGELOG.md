## [0.1.0] - 2025-12-17

### Added

- **Button Component**: `<nim-button>` with variant, size, and disabled support
- **Input Component**: `<nim-input>` with label, placeholder, and error state support
- **Design Tokens**: CSS custom properties for theming (`_variables.scss`)
- **Global Configuration**: `NimbusConfig` for library-wide settings
- **Project Structure**: Angular library with demo app for testing
- **Documentation**: `LICENSE.md` (MIT), `VERSIONING.md`, component checklist

### Infrastructure

- Angular 21 workspace setup
- ESLint and Prettier configuration
- Storybook integration (initial setup)
